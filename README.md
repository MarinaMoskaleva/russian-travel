# Проект: Путешествие по России

Проект посвящен путешествиям по территории стране. Представлен список мест, уникальные особенности которых делают их крайне интерсными для посещения, даны ссылки на источники информации о них. [Проект](https://marinamoskaleva.github.io/russian-travel/) выложен на GitHub Pages.

В проекте реализован механизм отзывчивой и адаптивной верстки, использована технология ”Grid Layout“ при построении сетки.
