# Проект: Путешествия по России
### Описание
Проект посвящен путешествиям по территории стране. Представлен список мест, уникальные особенности которых делают их крайне интерсными для посещения, даны ссылки на источники информации о них. 
### Страница проекта
[Путешествия по России](https://marinamoskaleva.github.io/russian-travel/)

### Технологии
- HTML
- CCS3
- Адаптивная верстка
- Методология БЭМ (файловая структура)
- Flexbox
- GRID
- Позиционирование
- Псевдоклассы
- Медиазапросы

### Инструкция для запуска
- Клонируйте проект ```git clone https://github.com/MarinaMoskaleva/russian-travel.git```.
